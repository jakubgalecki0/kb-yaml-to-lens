# Advanced Guides

Use the CLI to access bundled guide content:

- `kb-dashboard docs list-guides`
- `kb-dashboard docs guide <guide-id>`

The generated `llms-full.txt` replaces individual guide bodies with CLI references and short descriptions.

## Available Guide IDs

- `breaking-changes`
- `dashboard-decompiling-guide`
- `dashboard-style-guide`
- `esql-language-reference`
- `otel-dashboard-guide`
- `color-assignments`
- `legend-configuration`
- `esql-views`
